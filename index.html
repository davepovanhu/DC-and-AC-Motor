Voltage (V): Controls the input power to the motor.
Current (A): Controls the input current to the motor.
Speed (RPM): Controls the motor speed.
Frequency (Hz): Adjusts current frequency (AC motor).
Torque (Nm): Controls motor output torque.
Resistance (Ω): Adjusts motor resistance.
